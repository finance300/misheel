# UB Asset Management Web App (React + Vite)

This project is now a real React app codebase (not a static single-page HTML shell).

## Run in localhost

```bash
cd "/Users/ravdan/Documents/VS CODE/Misheel Web App"
npm install
npm run dev
```

Open:

- `http://127.0.0.1:5173/` for public website
- `http://127.0.0.1:5173/internal?role=board_member&lang=mn` for internal entry

## What is implemented

- Public corporate website structure (hero, about, strategy, team, timeline, contact)
- Mongolian / English toggle
- Login modal with role selection
- Internal portal entry route with role-aware rendering
